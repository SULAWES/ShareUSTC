# ShareUSTC - 学习资源分享平台

> 促进校内优质学习资源的共享与传承，打造互助学习社区



## 项目初心

ShareUSTC 诞生于一个简单的想法：让优质的学习资源在校园内自由流动。

作为大学生，我们常常面临这样的困境：
- 期末考试前四处寻找往年试题和复习资料
- 优质资源散落在QQ群或学长创建的网站(例如[我的科大](https://wdkd.feixu.site/))/Git仓库，难以检索和保存

- 辛苦整理的学习资料无法与更多同学分享
- 不清楚哪些资料真正有价值，下载到的资料质量良莠不齐

据我了解，很多大学会使用商业公司开发的小程序获取学习资料，但代价是观看广告/为资源付费，而我想创办一个社区，让优质学习资源无偿地在一届又一届学生中传承。

开发 ShareUSTC 的目的就是建立一个理想的学习资源分享平台：

- **资源共享**：支持上传、检索、预览、下载各类学习资料
- **质量保证**：通过评论、评分机制筛选优质内容
- **知识传承**：让学长学姐的经验惠及学弟学妹
- **互助氛围**：建立用户贡献激励体系，鼓励学生分享

如果网站的社区氛围能够建立起来，我想将 ShareUSTC 做成一个不单单能下载试卷，更能分享学习心得的网站！

## 功能特性

### 核心功能
- **资源管理**：支持上传、浏览、搜索、预览、下载各类学习资料
- **多格式支持**：PDF、Markdown、图片、文本等格式的在线预览（*PPT、DOC 正在开发中*）
- **评分评论**：多维度评分 （*正在开发中*）+ 评论互动
- **收藏夹**：创建个人收藏夹，支持打包下载
- **图床功能**：在线上传图片，获取 Markdown 引用链接
- **个人主页**：展示个人简介、上传资源、统计数据

### 用户角色
| 功能 | 游客 | 注册用户 | 管理员 |
|------|------|----------|--------|
| 浏览/搜索/下载资源 | ✅ | ✅ | ✅ |
| 上传资源 | - | ✅ | ✅ |
| 评分/点赞/评论 | ✅ | ✅ | ✅ |
| 创建收藏夹 | - | ✅ | ✅ |
| 个人主页 | - | ✅ | ✅ |
| 管理后台 | - | - | ✅ |

注册用户的实名认证正在开发中，届时部分互动功能将只对实名用户开放

### 管理后台

- 用户管理（禁用/启用账号）
- 资源、评论审核与管理（*开发中*）
- 全站通知发送
- 网站数据统计
